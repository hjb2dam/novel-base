내러티브 포지(NarrativeForge) 서비스 이용약관

최종 수정일: 2026년 3월

제 1조 (목적)

본 약관은 내러티브 포지(이하 "서비스")가 제공하는 웹소설 창작 보조, AI 설정 엔진, TRPG 시뮬레이션 및 커뮤니티 서비스의 이용과 관련하여, 서비스 제공자와 이용자 간의 권리, 의무 및 책임 사항, 기타 필요한 사항을 규정함을 목적으로 합니다.

제 2조 (서비스의 제공 및 포인트 정책)

  • 본 서비스는 창작자의 세계관 설정 관리, AI 기반 텍스트 생성 및 시나리오 보조 기능을 100% 무료로 제공합니다.
  • 서비스 내 AI 기능을 사용하기 위해 필요한 API 키(Groq 등)는 이용자가 직접 발급받아 연결해야 하며, 이에 따른 외부 API 서비스의 이용 규정은 해당 제공사의 약관을 따릅니다.
  • 원활한 서비스 이용을 위해 사이트 자체적인 '포인트(P)' 시스템을 운영합니다. 비로그인 게스트 사용자에게는 기본 10P가 지급되며, 정식 계정을 연동(로그인)한 회원에게는 1000P의 보너스 포인트가 지급됩니다.

제 3조 (보안 및 이용자의 의무)

  • 이용자가 입력한 외부 API 키는 이용자 기기의 브라우저 로컬 저장소(LocalStorage)에만 안전하게 보관되며, 본 서비스의 데이터베이스(서버)는 어떠한 경우에도 API 키를 수집하거나 저장하지 않습니다.
  • 이용자는 자신의 API 키 및 계정 정보를 안전하게 관리할 책임이 있으며, 이용자의 부주의로 인해 발생한 API 키 유출 및 그로 인한 모든 금전적, 정신적 피해에 대한 책임은 전적으로 이용자 본인에게 있습니다.
  • 이용자는 본 서비스를 불법적인 목적이나 타인의 권리를 침해하는 용도로 사용할 수 없습니다.

제 4조 (커뮤니티 검역 및 콘텐츠 관리)

  • 'Community Lounge' 및 'Writer's Board' 등 공개된 공간에 작성되는 모든 게시물은 건전한 창작 환경 조성을 위해 AI 시스템에 의한 자동 검역을 거칩니다.
  • 웹소설 집필, 작법, 팁, 세계관 설정 등 창작과 무관한 내용, 스팸, 혐오 표현, 타인 비방 등의 콘텐츠는 AI 검역에 의해 즉시 블락(Block) 처리됩니다.
  • 블락 처리된 게시물은 다른 이용자에게 노출되지 않으며, 시스템에서 제공하는 추천 보상(포인트 지급 등) 대상에서 영구적으로 제외됩니다.
  • 본인이 작성한 글 및 소설 데이터는 작성자 본인과 서비스 운영자만이 삭제할 수 있습니다.

제 5조 (저작권 및 면책조항)

  • 이용자가 본 서비스를 통해 직접 작성한 설정, 원고, 캐릭터 정보 등의 창작물에 대한 저작권은 이용자 본인에게 귀속됩니다.
  • 서비스 제공자는 무료로 제공되는 서비스의 특성상, 시스템 오류, 데이터 유실, 외부 API 연동 지연 등으로 인해 발생하는 어떠한 손해에 대해서도 법적 책임을 지지 않습니다. 중요한 원고 및 설정 데이터는 이용자가 직접 백업해야 합니다.
