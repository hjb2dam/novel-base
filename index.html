Terms of Service

제 1조 (서비스 목적 및 이용)

본 서비스 '노벨베이스'는 인공지능 기술을 활용하여 소설 작가의 설정 관리 및 원고 검수 보조를 목적으로 합니다. 모든 결과물은 AI 모델의 확률적 생성물이며, 최종적인 개연성 판단 및 저작물에 대한 책임은 사용자 본인에게 있습니다.

제 2조 (API 키 및 데이터 보안)

사용자가 입력한 Groq API 키 및 소설 설정 데이터는 암호화되어 사용자의 **로컬 브라우저(LocalStorage)**에만 저장됩니다. 노벨베이스는 서버 측에 어떠한 키 정보나 개인 설정을 수집하거나 저장하지 않습니다. 단, 브라우저 캐시 삭제 시 데이터가 소실될 수 있으므로 별도 백업을 권장합니다.

제 3조 (이용 제한 및 관리자의 책임 한계)

1. 사용자는 타인의 저작권을 침해하거나 불법적인 용도로 AI를 사용할 수 없습니다.
2. AI 검역 시스템에 의해 부적절한 게시물로 판단될 경우, 커뮤니티 포인트 지급이 거절될 수 있습니다.
3. 관리자는 API 제공사(Groq 등)의 장애로 인한 서비스 중단에 대해 책임을 지지 않습니다.

제 4조 (개인정보 보호)

본 서비스는 소셜 로그인을 통해 이메일 및 기본 프로필 정보를 수집하며, 이는 오직 회원 식별 및 게시판 운영 목적으로만 사용됩니다.

Ultimate API Guide

01

Groq Cloud 계정 생성

현존 최강의 속도를 자랑하는 AI 추론 엔진 Groq Console에 접속하여 구글 계정으로 간편하게 가입하세요.

02

비밀 API Key 발급

'Create API Key' 버튼을 클릭한 뒤 이름을 입력하면 gsk_...로 시작하는 키가 생성됩니다. 이 키는 현금과 같으니 절대 노출하지 마세요.

03

안전한 로컬 적용

복사한 키를 메인 화면 상단 'Private Key' 입력창에 붙여넣으세요. 브라우저가 종료되어도 작가님의 PC에만 암호화되어 안전하게 유지됩니다.
