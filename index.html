Ultimate API Guide

01

Groq Console 접속 후 'Create API Key'를 눌러 gsk_... 키를 발급받으세요.

02

발급된 키를 복사하여 메인 화면의 'Private Key' 입력창에 넣으세요. 이 키는 작가님의 개인 비서 비용을 지불하는 용도와 같습니다.

03

노벨베이스는 서버에 키를 저장하지 않습니다. 오직 작가님의 브라우저에만 암호화되어 저장되니 안심하고 사용하세요.
