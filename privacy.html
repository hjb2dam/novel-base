내러티브 포지(NarrativeForge) 개인정보처리방침

최종 수정일: 2026년 3월

1. 수집하는 개인정보 항목

내러티브 포지는 회원가입, 원활한 클라우드 동기화 및 서비스 제공을 위해 아래와 같은 최소한의 개인정보를 수집하고 있습니다.

  • 필수 수집 항목: 이메일 주소, 비밀번호, 서비스 내 닉네임
  • 자동 수집 항목: 서비스 이용 기록(작성한 소설 설정, 원고 데이터, 커뮤니티 게시글 등) 및 보유 포인트

2. 개인정보의 이용 목적

수집된 개인정보는 다음의 목적을 위해서만 활용됩니다.

  • 회원 식별 및 로그인, 클라우드 데이터(소설, TRPG 로그 등) 동기화 지원
  • 커뮤니티 게시판 이용 시 작성자 식별 및 추천(포인트) 시스템 운영
  • 서비스 부정 이용 방지 및 비인가 사용 통제

3. [중요] API 키 및 민감 데이터 처리 방침

  • 내러티브 포지는 AI 텍스트 생성을 위해 이용자가 입력한 외부 API 키(Groq 등)를 일절 수집하거나 중앙 서버에 저장하지 않습니다.
  • 이용자의 API 키는 오직 이용자 본인 기기의 브라우저 로컬 저장소(LocalStorage)에만 암호화되지 않은 텍스트 형태로 임시 보관되며, API 호출 시 이용자의 브라우저에서 AI 제공 서버로 직접 전송됩니다.

4. 개인정보의 보유 및 파기

이용자의 개인정보 및 클라우드 연동 데이터(Supabase 기반)는 이용자가 회원 탈퇴를 요청하거나 개인정보 삭제를 요구할 때까지 보관됩니다. 회원 탈퇴 시, 이용자의 데이터베이스 기록은 복구 불가능한 방법으로 지체 없이 파기됩니다.

5. 쿠키(Cookie) 및 로컬 스토리지 사용

본 서비스는 이용자에게 개인화된 환경(다크모드/라이트모드 테마, 언어 설정, API 키 임시 보관, 투표 중복 방지 등)을 제공하기 위해 브라우저의 로컬 스토리지(LocalStorage)를 사용합니다. 이는 이용자의 컴퓨터를 식별할 뿐 이용자 개인을 식별하지는 않습니다.
